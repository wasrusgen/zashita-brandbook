# ЗАЩИТА — Бренд-бук

Руководство по фирменному стилю суббренда **ЗАЩИТА** (@wasrusgen1) — юридическая экспертиза и сопровождение сделок.

## Содержимое

- `brandbook-zashita.html` — бренд-бук (логотип, цвета, типографика, UI-компоненты, токены)
- `tokens.css` — дизайн-токены (CSS-переменные)
- `logos/logo-wasrusgen1-real.svg` — знак @wasrusgen1

## Просмотр

Открыть `brandbook-zashita.html` в браузере. Шрифты (Inter, Montserrat) подгружаются из Google Fonts.

## Бренд

- Акцент — Бордо `#9F1239`, глубокий `#3B0212`
- Логотип-шрифт — Montserrat 700 (только знак/суббренды)
- UI-шрифт — Inter 300–800
- Палитра Бордо: 950 → 50
