Важные правила ЗАЩИТА
1. Montserrat 700 — только для логотипа. В UI не применять.
2. Суббренд ЗАЩИТА — юридический кабинет: экспертиза договоров, контроль сроков, сопровождение сделок.
3. Logo gray #5B5B5B — цвет суббренда на светлом. На тёмном фоне — rgba(255,255,255,0.92).
4. Все цвета — только через токены. Не хардкодить HEX в компонентах.
5. Sidebar ЗАЩИТА: #0C1929 (Бордо-dark tint), не чистый чёрный. Active nav border: #9F1239.
